## 1. 流程层 4 件根治

### 1.1 H25 chapter-aware 修订（Round 20.1 H25 P0 升级的误伤根治）

**根因**：Round 20.1 加了 H25 P0 fail "连续 8 章无决策钩"，但取的是 `chs[-8:]`（最近 8 章）不区分当前 chapter。Ch3 polish 时 H25 检查 Ch5-12 全无决策钩 → P0 fail → polish_cycle 拒绝 commit Ch3。这是误伤。

**根治位置**：`hygiene_check.py:1564`

**修法**：H25 P0 仅在"当前 chapter ≥ chs[-1]"（即 polish 当前最新章或更新章）时触发。早期章节 polish 不被未来章节状态阻断。

**实战验证**：Ch3 polish 后 H25 不再误报，commit `a25d4b1` 成功落地。

### 1.2 chapter_meta.thrill_score 白名单（Bug 1 根治）

**根因**：reader-thrill-checker 6 子维度结构需要嵌套 dict 写入 chapter_meta，但 set-chapter-meta-field 白名单不含 thrill_score。

**根治位置**：`state_manager.py:1730`

**修法**：`CHAPTER_META_FIELD_WHITELIST` 加 `thrill_score`。

**实战验证**：Ch3/Ch4/Ch6/Ch2 thrill_score 全部成功通过 CLI 落库。

### 1.3 dialogue_ratio_override Ch3 配置豁免（流程层根治）

**根因**：Ch3 是空间种田激活章，主角独自在空间催种，dialogue_ratio 0.053 < 0.20 硬线。post_draft_check fail。

**根治位置**：`末世重生-.../.webnovel/post_draft_config.json`

**修法**：`dialogue_ratio_override_chapters: [3]`（项目级 override）。

**实战验证**：Ch3 polish 时 DIALOGUE_RATIO 不再 fail，post_draft 全过。
